# LumiformApp
### Architecture

* Min iOS target 16
* Modern Swift concurrency
* Offline mode support with Core Data
* MVVM-C architecture
* Light and Dark mode support

### Installation

Clone the repo
   ```sh
   git clone https://github.com/emrebingor/LumiformApp.git
   ```
